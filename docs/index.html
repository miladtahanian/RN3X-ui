✨ Features

همه چیز برای مدیریت پنل 3X-UI در هر زمان و مکان

Everything you need to manage your 3X-UI panel on the go.

📊

داشبورد

Dashboard

مانیتورینگ لحظه‌ای سرور — CPU، RAM، دیسک، Swap، آپتایم، وضعیت Xray و میانگین بار

Real-time server monitoring — CPU, RAM, Disk, Swap, Uptime, Xray status, and average load.

📈

آمار ترافیک

Traffic Analytics

نمایش ترافیک آپلود/دانلود با نمودارهای پیشرفت و اعداد دقیق

Upload/download traffic visualization with progress bars and detailed numbers.

👥

کاربران آنلاین

Online Users

مشاهده کاربران متصل به سرور در لحظه

See who's connected to your server in real time.

👤

مدیریت کاربران

Client Management

لیست، جستجو، فیلتر، حذف کاربران. ریست ترافیک، مشاهده جزئیات مصرف و اطلاعات کاربر

List, search, filter, delete clients. Reset traffic, view detailed usage and metadata.

🔗

جزئیات کاربر

Client Details

تفکیک ترافیک، شناسه، محدودیت IP، تاریخ انقضا، Sub ID و لینک‌های اشتراک‌گذاری

Traffic breakdown, ID, IP limit, expiry, Sub ID, and shareable connection links.

🌐

مدیریت Inboundها

Inbound Management

مشاهده تمام inboundها با نشان پروتکل (VLESS, VMESS, Trojan, Shadowsocks و ...)، فعال/غیرفعال کردن، حذف

View all inbounds with protocol badges. Enable/disable toggle, delete. Supports VLESS, VMESS, Trojan, Shadowsocks & more.

⚙️

تنظیمات پنل

Panel Settings

اطلاعات اتصال، تنظیمات پنل، راه‌اندازی مجدد، تغییر مدیر، مدیریت ربات تلگرام و تنظیمات اشتراک

Connection info, panel configuration, restart panel, change admin, manage Telegram bot & subscription settings.

🌍

دو زبانه

Multi-language

پشتیبانی کامل از فارسی و انگلیسی با تغییر خودکار جهت RTL/LTR

Full support for English and Persian (فارسی) with automatic RTL/LTR switching.

🌙

تم تیره

Dark Theme

رابط کاربری تیره و مدرن با انیمیشن‌های روان و پالت رنگی精心 طراحی شده

Modern, eye-friendly dark UI with smooth animations and a carefully designed color palette.
