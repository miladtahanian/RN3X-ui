## Features

- **Dashboard** — Server status (CPU, RAM, Disk, Swap, Uptime), Xray status, avg load (CPU/RAM/DISK), live traffic, total data (all clients), server address, online users
- **Client Management** — List, search, filter, delete, reset traffic, detailed usage
- **Inbound Management** — View all inbounds, enable/disable, delete
- **Settings** — Organized into separate pages: Panel, Subscription, API Token, Operations, About
- **API Token** — Bearer token support for API requests
- **Multi-language** — English & Persian with RTL/LTR support
- **Dark Theme** — Modern dark UI with smooth animations
